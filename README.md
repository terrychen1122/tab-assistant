# Tab Assistant

Manifest V3 Chrome extension for:

- searching tabs in the current window
- recommending low-value tabs to close
- suggesting tab groups from title and URL similarity

## Load in Chrome

1. Open `chrome://extensions`.
2. Enable Developer mode.
3. Click `Load unpacked`.
4. Select `/Users/xuchen/Developer/TabAssistant`.

## Run tests

```bash
npm test
```
